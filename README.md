# 🌸 Asiya's DSA Journey

An interactive **100-Day Data Structures & Algorithms roadmap** built to make learning DSA fun, structured, and motivating.

Whether you're preparing for coding interviews or simply building problem-solving skills, this tracker helps you stay consistent with daily goals, progress tracking, and an XP-based leveling system.

## 🌐 Live Demo

**🔗 https://asiyabeig.github.io/dsa_roadmap/**

---

## ✨ Features

* Structured **100-Day DSA Roadmap**
* 130+ carefully selected LeetCode problems
* XP system with level progression
* Daily topic checklists
* Streak tracker
* Automatic progress saving using `localStorage`
* Soft pastel UI (Pink • Lavender • Green)
* Custom animated cursor
* Responsive design for desktop and mobile
* Fully offline — no backend required

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/asiyabeig/dsa_roadmap.git
```

Open the project folder and simply launch:

```text
index.html
```

No installation, build tools, or dependencies required.

## 🎯 Why I Built This

Most DSA sheets feel overwhelming or become difficult to stay consistent with. I wanted a roadmap that felt more like a game than a checklist.

This project combines:

* structured learning,
* gamification,
* progress tracking,
* and a clean aesthetic

to make practicing DSA enjoyable every day.
