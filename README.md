# Asiya's DSA Journey

A personal interactive DSA tracker — 100 days, all topics, fully offline.

## Features
- 100-day structured roadmap (C++ → DP)
- Problem bank with 130+ LeetCode problems
- XP system with level progression
- Topic checklist per phase
- Streak tracking
- Progress saved in localStorage
- Whimsical soft pink/green/lavender design
- Custom cursor

## How to use
Just open `index.html` in any browser. No build step, no dependencies.

## Push to GitHub Pages
```bash
git init
git add .
git commit -m "init: dsa journey"
git remote add origin https://github.com/asiyabeig/dsa_roadmap.git
git push -u origin main
```

Then go to Settings → Pages → Source: main → `/root`.
